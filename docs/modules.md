# Modules Overview

## inspection.py
Used for early data diagnosis.

## missing_values.py
Handles missing values using SimpleImputer.

## outliers.py
Removes outliers using IQR method.

## encoding.py
Supports:
- Label Encoding
- One-Hot Encoding

## scaling.py
Supports:
- StandardScaler
- MinMaxScaler

## splitting.py
Splits dataset into train and test sets.
